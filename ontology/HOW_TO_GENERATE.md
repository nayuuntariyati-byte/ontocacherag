# Ontology Files

This directory will contain the generated OWL ontology files after running `build_ontology.py`.

## Expected Files (after generation)

| File | Generated By | Description |
|---|---|---|
| `arsipdataset_ontology.ttl` | `scripts/build_ontology.py` | Turtle format ontology |
| `arsipdataset_ontology.owl` | `scripts/build_ontology.py` | OWL/XML format ontology |
| `ontology_stats.json` | `scripts/build_ontology.py` | Statistics matching paper Section 5.2 |

## How to Generate

After preprocessing your ArsipDataset CSV (see `data/README.md`), run:

```bash
python scripts/build_ontology.py \
    --input data/arsipdataset_normalized.csv \
    --output-dir ontology/ \
    --verbose
```

## Expected Statistics

The generated `ontology_stats.json` should report:

```json
{
  "tbox": {
    "total_classes": 131,
    "by_level": {"L1": 1, "L2": 1, "L3": 5, "L4": 124}
  },
  "properties": 2,
  "abox": {
    "instances": 614,
    "in_corpus_revocations": 42,
    "out_corpus_revocations": 6
  },
  "total_triples": 1901
}
```

These numbers correspond directly to Table 3 in paper Section 5.2.

See [`README.md`](README.md) for full ontology specification and SPARQL query examples.
